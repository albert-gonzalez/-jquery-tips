jQuery Tips and Examples

Selectors

Jquery has a search engine which allows us make queries with CSS selectors to retrieve DOM elements. The following tips show how to use them properly.

Avoid jQuery Selector Extensions

jQuery Selector Extensions cannot take advantage of the performance boost provided by the native querySelectorAll(). Try to use native css selectors if is possible.
